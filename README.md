# Personal Portfolio

Personal portfolio website built to showcase my projects, skills and experience as a web developer.

The goal of this project is to present my work in a clear, professional and functional way, focusing on clean code, usability and modern web practices.

## ✨ Features

- One-page layout
- Fully responsive design (mobile, tablet and desktop)
- Smooth scrolling navigation
- Projects section with live demos and source code
- Skills organized by category
- Contact form with real email sending (EmailJS)
- Modern UI with subtle animations
- Floating header with blur effect on scroll

## 📬 Contact Form

The contact form is powered by EmailJS, allowing messages to be sent directly to my email without the need for a backend server.

To use it in your own project:

Create an EmailJS account

Configure an email service and template

Add your keys as environment variables

Update the EmailJS configuration in Contact.jsx

## 🌍 Deployment

The project is deployed using Vercel.

## Deployment steps:

Push the repository to GitHub

Import the project into Vercel

Select Vite as the framework

Build and deploy
